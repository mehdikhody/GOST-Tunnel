# GOST-Tunnel

Tunnel your server via GOST

# Install

```bash
bash <(curl -Ls https://raw.githubusercontent.com/mehdikhody/GOST-Tunnel/master/install.sh)
```

# Update

```bash
sudo gtctl update
```

# Start / Stop / Restart / Status

```bash
sudo gtcl start
sudo gtcl stop
sudo gtcl restat
sudo gtcl status
```

# Uninstall

```bash
sudo gtctl uninstall
```
